## Preparación

### Instalar paquete sfml

En Linux:
sudo apt update
sudo apt install libsfml-dev

En Windows:
vcpkg install sfml
vcpkg integrate install

# Como ejecutar

- Usa "make run" para compilar
- Usa "make runAll" para compilar y ejecutar
